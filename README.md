# DOME-HUB

Home base for all projects, platforms, software, compute, agents, AI models, knowledge bases, databases, and codebases.

## Structure

```
DOME-HUB/
├── projects/       # Individual projects
├── platforms/      # Platforms & products
├── software/       # Software packages & tools
├── compute/        # Infrastructure & compute configs
├── agents/         # AI agents
├── models/         # AI models & fine-tunes
├── kb/             # Knowledge bases
├── db/             # Databases & data stores
├── codebase/       # Shared libraries & core code
└── scripts/        # Automation & utility scripts
```

## Environments

- `dev` — local development
- `prod` — production
