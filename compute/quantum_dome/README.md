# compute/quantum_dome — Quantum Computing Module

Local quantum computing lab. All frameworks pre-installed.

| File | Purpose |
|------|---------|
| `core.py` | Quantum circuit primitives |
| `scheduler.py` | Workload scheduling |
| `pool.py` | Compute pool management |
| `memory.py` | Quantum memory management |
| `profiler.py` | Performance profiling |
| `example.py` | Usage examples |

Frameworks: Qiskit, PennyLane, Cirq, QuTiP, PyQuil, Amazon Braket
